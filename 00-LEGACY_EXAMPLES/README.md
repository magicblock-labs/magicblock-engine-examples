# Backward Compatibility Examples

The programs in this folder are intentionally kept for backward compatibility.
They preserve the pre-Anchor 1.0 versions of examples that were migrated in the
main repository to Anchor 1.0.x.

Use these examples when you need to:

- compare behavior against the older Anchor 0.32.1 implementations;
- support clients, tests, or deployments that have not moved to Anchor 1.0 yet;
- reference the previous MagicBlock integration patterns while upgrading.

For new development, start from the top-level examples in this repository. Those
are the current versions of the programs after the Anchor 1.0 upgrade.
