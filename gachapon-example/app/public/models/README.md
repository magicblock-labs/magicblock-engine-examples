# Model Placement

Put the exported Blender model here:

```text
gachapon-example/app/public/models/gachapon.glb
```

The frontend loads it from:

```text
/models/gachapon.glb
```

Expected object names:

- `Machine`
- `Knob`
- `Ball`

The knob should have its object origin on the axle you want to rotate around.
